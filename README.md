A cross-platform desktop drawing application built using **Java Swing** and **AWT**. This project provides a user-friendly interface for creating, editing, and saving digital sketches with various tools and customization options.

## âœ¨ Features

- **Drawing Tools**: 
  - **Brush**: Smooth freehand drawing.
  - **Shapes**: Draw perfect Lines, Rectangles, and Ovals.
  - **Eraser**: Easily remove mistakes from the canvas.
- **Customization**:
  - **Color Picker**: Choose any color via the standard `JColorChooser`.
  - **Brush Thickness**: Adjust stroke size using a dynamic slider (1px - 50px).
  - **Fill Mode**: Toggle between outlined and solid shapes.
  - **Dotted Lines**: Create dashed/dotted strokes for artistic effects.
- **Canvas Management**:
  - **Undo**: Revert your last action.
  - **Clear All**: Wipe the canvas clean (with a safety confirmation).
- **File Handling**:
  - **Save**: Export your artwork to your local machine.
  - **Open**: Import existing images to use as a background or base.
